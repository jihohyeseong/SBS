# SBS Online Bookmarket
기존의 오프라인(서점)에서 책을 구매하는 소비자를 위해 간편하고 빠른 
서비스를 제공하기 위해 온라인 북 쇼핑몰을 제공, 댓글 요약, 책읽기 음성ai, 
책 내용 텍스트를 통해 배경이미지 제공 등 편리한 기능 제공

## 개발자소개
